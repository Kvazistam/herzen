## Задание 1. 
С помощью пакета Cython преобразовать программу по факторизации числа методом ферма.

------
Были созданы 2 файлы. ferma.py - модуль по факторизации методом ферма на чистом Python. ferma_cy.pyx - текстовый файл повторяющий функционал ferma.py с незначительными изменениями в стиле Cython. А именно были добавлены объявления типы переменных, к сожаление числа в тестовом примере превышают 10^20, поэтому явно объявить все перменные не удалось, тк тип long long только до 10^18, из-за этого происходило переполнение.
